# src/components

可复用 UI 组件目录，按功能模块划分子目录。

## 目录结构

```
components/
├── common/      通用组件（按钮、图标、空状态等）
├── layout/      布局组件（页头、页脚、侧边栏等）
├── movie/       电影相关组件（卡片、列表项、评分等）
├── order/       订单相关组件（订单项、时间线等）
└── business/    业务组件（筛选器、搜索框等）
```

## 使用规范

- 组件名使用 PascalCase
- 每个组件一个 `.vue` 文件
- 优先使用 Naive UI 组件作为基础
- 复杂组件拆分为子组件
